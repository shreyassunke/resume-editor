# V1 — SWE / Big Tech

## Purpose

The default version for software engineering applications. It leads with technical
depth and architecture ownership: ATLAS (RAG/AI systems) and HUMN AI (full-stack
architecture, CI/CD, scalability). Use this for most SWE intern, backend, and
full-stack applications.

## Targets

- **Roles:** Software Engineer Intern, Backend Engineer, Full-Stack Engineer
- **Companies:** Amazon, Microsoft, Tableau, mid-stage AI startups

## ATS keywords this version leads with

algorithms, data structures, REST APIs, CI/CD, system design, Git, Python,
JavaScript, React, Node.js, scalability, distributed systems

## Bullet-ordering rationale

- **HUMN AI** leads with architecture/CI/CD, then the full-stack stack + perf/cost
  numbers, then React/REST API breadth, then Stripe last (Stripe is breadth here,
  not the headline for general SWE roles).
- **ATLAS** leads with the RAG pipeline, then the full-stack/vector-DB architecture,
  then the in-progress knowledge-graph work.
- **Skills** put languages and frameworks front and center, with algorithms / data
  structures / system design / distributed systems in Concepts for ATS coverage.
